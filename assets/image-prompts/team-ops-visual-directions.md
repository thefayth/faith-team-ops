# Faith Team Ops Image Prompts

These prompts are public-safe Canva or image-generation prompts for expanding the existing visual package. They do not describe private implementation details.

## Hero Image

Create a polished public hero image for Faith Team Ops, a private WordPress-based operations platform for distributed teams. Show a serious but humane operations dashboard with role scope, check-ins, support requests, audit receipts, and mobile-first coordination. Use deep green, teal, white, slate, and restrained gold. Avoid generic SaaS stock styling, purple gradients, decorative orbs, fake private data, credentials, or excessive marketing polish.

## GitHub Banner

Create a wide GitHub repository banner for Faith Team Ops. The banner should communicate public surface, private engine, receipts always. Include abstract dashboard panels, a WordPress portal entry, role scope, audit receipt lines, and a subtle global coordination feel. Use clean typography, high contrast, and accessible spacing.

## Social Card

Create a 1200 x 630 social preview card for Faith Team Ops. Show the project name, short line "Private operations. Public receipts.", and visual hints of Signal Desk, Field Command, and Care Ledger without exposing source code or private records.

## Project Icon

Create a square icon for Faith Team Ops using a shield or operations mark, deep green base, restrained gold accent, and the letters FTO. Keep it legible at small sizes and avoid generic clip art.

## Workflow Illustration

Create a public-safe workflow illustration showing: team members enter a WordPress portal, role and country scope guide submissions, admin review creates receipts, and public GitHub/WordPress materials explain the project while protected systems stay private.
