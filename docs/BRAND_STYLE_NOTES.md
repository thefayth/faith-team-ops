# Brand Style Notes

## Brand Feel

Faith Team Ops should feel bold, intelligent, strategic, human, and operational. It should not feel like a generic SaaS template, fundraising landing page, or decorative community site.

## Shared Visual Language

- Deep green for trust, authority, and grounded operations.
- Teal for coordination, movement, and field command.
- White and soft neutral surfaces for legibility.
- Restrained gold for priority, receipts, and ownership.
- Slate text for calm secondary information.
- Compact cards and rows for operational scan speed.
- Mobile-first layouts for low-friction field use.

## Direction 1: Signal Desk

Use when the first priority is daily operations clarity.

- Deep green navigation.
- White cards.
- Clear check-in and support actions.
- Fast-scanning task and audit rows.
- Best for day-one WordPress implementation.

## Direction 2: Field Command

Use when country coordination and live operations need to feel central.

- Teal command surface.
- Country scope shown prominently.
- High-contrast event and audit rail.
- Dense but organized mobile dashboard.
- Best for active country-lead coordination.

## Direction 3: Care Ledger

Use when support, dignity, accountability, and trust need to lead the experience.

- Black, green, and gold navigation.
- Warm support language.
- Ledger-style audit rows.
- Care-forward iconography.
- Best for moderation, support, pastoral/team care, and accountability.

## Avoid

- Generic blue enterprise palettes.
- Purple gradient themes.
- Decorative blobs or stock illustrations.
- Oversized marketing heroes inside the private portal.
- Map-first visuals unless map data is real.
- Flags or token country imagery as default decoration.
- Fear-heavy security theater.
