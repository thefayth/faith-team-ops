# Security

Faith Team Ops is a private operations platform with a public project surface.

## Do Not Disclose

Do not open public issues, pull requests, discussions, screenshots, or comments containing:

- Credentials, tokens, API keys, cookies, or environment values.
- Private WordPress admin paths or protected deployment details.
- Team records, support requests, private media, voice notes, or internal audit data.
- Unpublished strategy, legal/admin records, medical records, benefits records, or family information.
- Internal prompts, agent instructions, automation workflows, or private infrastructure diagrams.

## Reporting

If you believe you found a security issue, use the private contact path on FaithCheltenham.com instead of posting details publicly.

Include only the minimum safe summary needed to identify the issue. Do not include secrets or sensitive user data in the initial report.

## Public Materials

The public repository documents the system boundary and product shape. It intentionally excludes source code, runtime configuration, private workflows, and protected operational records.
