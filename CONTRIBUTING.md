# Contributing

Faith Team Ops is not an open-source project and this repository is not a source-code collaboration space.

Public feedback may be useful when it concerns:

- Documentation clarity.
- Public/private boundary wording.
- Accessibility of public explanations.
- Visual communication of workflows.
- High-level product questions that do not expose private operations.

Do not submit private code, credentials, internal prompts, private screenshots, team data, or reconstructed implementation details.

Pull requests are not expected unless Faith explicitly requests a public documentation contribution.
